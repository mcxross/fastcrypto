# lib

Kotlin Multiplatform bindings for `fastcrypto` generated by Gobley + UniFFI.

## Overview

- Rust exports live in `kotlin/lib/src/commonMain/rust` and use `#[uniffi::export]` + `uniffi::setup_scaffolding!()`.
- Gobley Gradle plugins build Rust and generate Kotlin bindings automatically.
- Generated Kotlin package: `xyz.mcxross.fastkrypto`.

## Build

From `kotlin/`:

```sh
./gradlew :lib:publishToMavenLocal
```

Gobley will build the Rust crate defined by `kotlin/lib/Cargo.toml` and generate Kotlin
bindings as part of the Gradle build.

## Security notes

- Private key inputs are zeroized after signing/recovery operations.
- Inputs are validated for expected byte lengths before use.
- Hashes and HMAC/HKDF use fastcrypto defaults.
